Air Hockey Solitaire - PWA build

- Includes: manifest.json, service-worker.js, icon-192.png, icon-512.png
- Deploy the entire folder to GitHub Pages (or any static host).
- Visit once online to cache assets for offline play.
- iOS: Share -> Add to Home Screen
- Android/Chrome: Install from the browser menu.

If you update files later, bump CACHE_NAME in service-worker.js to force-refresh caches.
